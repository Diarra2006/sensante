# SenSante

Assistant de pre-diagnostic medical pour le Senegal.

## Description

SenSante utilise le Machine Learning pour aider au pre-diagnostic des maladies couraantes (paludisme, grippe, typhoide) a partir des symptomes du patient.

## Structure du projet

- 'data/' : Donnes patients (CSV)
- 'models/' : Modele ML serialise
- 'api/' : API FastAPI
- frontend/' : Interface web
- 'notebooks/' : Scripts d'exploration
